# Vidly Genres API

A simple RESTful API built with Node.js and Express for managing movie genres.

## Features
- Get all genres
- Get a genre by ID
- Create a new genre
- Update an existing genre
- Delete a genre
- Input validation using Joi

## Tech Stack
- Node.js
- Express.js
- Joi
- NPM

## How to Run Locally
```bash
npm install
nodemon index.js

Server runs on:
http://localhost:3000
